# OrangeHRM Manual Testing Project

## Project Overview

This repository contains a complete Manual Testing project performed on the OrangeHRM Web Application.

The project follows the Software Testing Life Cycle (STLC) from Requirement Analysis to Test Closure and includes all major QA deliverables such as Test Plan, Test Scenarios, Test Cases, Bug Reports, and Test Summary Report.

---

## Application Under Test

OrangeHRM Open Source HR Management System

Modules Tested:

* Login
* Search
* My Info
* Admin
